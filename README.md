## 📌 1. 프로젝트 주요 기능 
- **리뷰 중심 커뮤니티**: 단순 별점을 넘어 영화별 리뷰와 자유 게시판을 통한 사용자 소통 강화
- **개인화된 관리**: 마이페이지를 통한 본인의 예매 내역 및 리뷰/좋아요 영화 관리
- **데이터 기반 관리**: 통계 기능을 통해 사이트 활성도를 한눈에 파악하는 관리자 대시보드

## ✨ 3. Key Functions (상세 기능)

### 🎥 커뮤니티 & 리뷰 (Core)
- **영화별 리뷰 CRUD**: 각 영화 상세 페이지에서 독립적인 리뷰 작성, 수정, 삭제 기능
- **자유 게시판**: 사용자 간 영화 추천 및 정보 공유를 위한 게시판 제공
- **좋아요 시스템**: 선호하는 영화를 찜하고 마이페이지에서 모아보기

### 🎫 예매 & 마이페이지
- **실시간 예매**: 영화 선택 및 좌석 배치도를 활용한 예매 프로세스
- **마이페이지**: 
  - 예매 내역 확인 및 취소
  - 내가 쓴 리뷰 목록 및 좋아요한 영화 관리

### 📊 관리자 서비스 (Admin)
- **콘텐츠 관리**: 상영 영화 정보 및 게시글/리뷰 관리 권한
- **사용자 통계**: 전체 가입자 수, 영화별 예매율 등 사이트 운영 통계 시각화
